## Getting Started

```sh
git clone git@github.com:threesigmaxyz/mezzanotte-yakyuken.git
cd mezzanotte-yakyuken
make update
make install
```

## Usage

After installing this branch, run with "make anvil" and then "made deploy-anvil" to check the deployment costs in this implementation without improvements
